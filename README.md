# Autonomi-extra

Useful Rust code for Autonomi network, maintained by community.

Started with a discussion on Autonomi Forum:
https://forum.autonomi.community/t/high-level-datatypes-library/41694

## Contributing

* All changes are made through PRs, preferably from private forks.
* Each PR is reviewed by 2 members chosen randomly and merged when accepted.
* Larger contributions can be proposed through issues, or PRs with just documentation draft, and discussed further before implementation.
